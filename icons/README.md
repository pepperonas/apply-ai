# Icons

Placeholder icons for the Chrome Extension.

To create proper icons:

1. Create a 128x128px icon with your logo/design
2. Use an image editor to resize to:
   - 16x16px → icon16.png
   - 48x48px → icon48.png
   - 128x128px → icon128.png

Or use an online tool like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

For now, you can use any placeholder images or create simple colored squares.
